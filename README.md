# Dotfiles

My personal dotfiles configuration repository.

## About

This repository contains my personal configuration files (dotfiles) for various tools and applications I use in my development environment.

## Installation

1. Clone this repository:
```bash
git clone https://github.com/osLeonardo/dotfiles.git
cd dotfiles
```

2. Review the configurations and customize them to your needs
3. Create symbolic links or copy the files to their appropriate locations

## Contents

Configuration files for various development tools and applications.
